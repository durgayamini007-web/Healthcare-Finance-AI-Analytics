AI-Powered Healthcare Financial Analytics Platform

## Overview

This project demonstrates the development of a Healthcare Financial Analytics Platform using Databricks, PySpark, and Spark SQL. The solution integrates healthcare patient data with insurance claims data to analyze operational performance, financial trends, and key business metrics.

## Business Objective

Healthcare organizations generate large volumes of patient and claims data. This project combines healthcare and insurance datasets into a unified analytics model to identify trends, monitor KPIs, and support data-driven decision-making.

## Project Workflow

1. Uploaded healthcare and insurance claims datasets into Databricks.
2. Loaded and processed data using PySpark DataFrames.
3. Integrated healthcare and financial records using patient-level joins.
4. Performed data transformation and aggregation using PySpark and Spark SQL.
5. Developed visualizations and KPI dashboards.
6. Generated operational and financial insights from integrated datasets.
